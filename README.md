# Vulkan

This repository is a follow-along of the [Vulkan-Tutorial](https://vulkan-tutorial.com/) and the goal of the project is to draw a triangle to the screen. 

## Application Accomplishments So Far

* Creates a Vulkan Instance
* Sets up Validation Layers
* Creates a Physical Device 
* Creates Graphics and Presentation Queue Families
* Creates Logical Device
* Creates a Window Surface
* Creates a Swap Chain
